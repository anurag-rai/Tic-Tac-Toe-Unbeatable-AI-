Following are the main features of the project:

  • Pages are made with a combination of ASP.NET pages, HTML and CSS as front-end, C# as back-end and ASP.NET as framework.
  • The user data is stored using SQL Tables by sending queries. Each user will have a unique username.
  • Passwords are encrypted as MD5 hashes.
  • Cookie is created with every login and is deleted when the user logs out.
  • Sending game/score url links to some-else has been disabled by using cookies.
  • Created a perfect Tic-tac-toe AI using methods derived from the Min-Max algorithm.
  • Sounds/Audio effects have been associated with some button features and game-play.
